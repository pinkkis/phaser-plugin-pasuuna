Phaser Pasuuna Plugin

The Pasuuna player plays SoundTracker, ProTracker and FastTracker mods. This plugin wraps it for use in Phaser 3 projects.

A generic Pasuuna example is available on that library's page: Pasuuna Player

The example below demostrates basic playback and event calls, and how volume controls can be separate for Pasuuna and the rest of Phaser's main volume.
